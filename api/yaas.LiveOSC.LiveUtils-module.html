trackVolume(track, volume=None)

source code 

Gets/Changes the volume of track (track)

If (volume) is specified, changes the volume of track number (track) to (volume), a value between 0.0 and 1.0.

trackPan(track, pan=None)

source code 

Gets/Changes the panning of track number (track)

If (pan) is specified, changes the panning to (pan). (pan) should be a value between -1.0 to 1.0

trackSend(track, send=None, level=None)

source code
